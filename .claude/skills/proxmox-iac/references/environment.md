# Environment Reference

Single source of truth for all environment values used in code generation.
Never hardcode values not listed here — ask the developer if something is missing.

## Networking

| Name | Purpose | Bridge |
|---|---|---|
| `vmbr0` | Main VM network — all VMs use this | Primary |

All VMs get IPs via DHCP from the home router. Expected range: `192.168.1.101` onwards.

## File Locations

| File | Purpose |
|---|---|
| `terraform/main.tf` | VM resources |
| `terraform/variables.tf` | All configurable values |
| `terraform/outputs.tf` | Post-apply output (IPs, URLs) |
| `terraform/inventory.tpl` | Template for Ansible inventory |
| `ansible/playbook.yml` | Main playbook |
| `ansible/inventory.ini` | Auto-generated — do not edit by hand |
| `ansible/roles/webserver/` | Nginx + hello world role |
| `autoscale/autoscale.sh` | CPU-based autoscaler |
| `dashboard/index.html` | Monitoring UI |
